# Real-Time Object Detection using YOLOv8

## 📌 Overview
This project performs real-time object detection using YOLOv8.

## 🚀 Features
- Image Detection
- Video Detection
- Webcam Detection
- Bounding Boxes
- Confidence Scores
- CSV Detection Logs

## 🛠 Technologies
- Python
- OpenCV
- YOLOv8
- PyTorch

## ▶ Installation

pip install -r requirements.txt

## ▶ Run Image Detection

python detect_image.py

## ▶ Run Video Detection

python detect_video.py

## ▶ Run Webcam Detection

python webcam_detection.py

# 🌐 Run Streamlit Web App

## Start Streamlit App

```bash
streamlit run app.py
```
