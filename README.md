# Real-Time Object Detection using YOLOv8

## 📌 Overview
This project performs real-time object detection using YOLOv8.

## 🚀 Features
- Image Detection
- Video Detection
- Webcam Detection
- Bounding Boxes
- Confidence Scores
- CSV Detection Logs

## 🛠 Technologies
- Python
- OpenCV
- YOLOv8
- PyTorch

## ▶ Installation

pip install -r requirements.txt

## ▶ Run Image Detection

python detect_image.py

## ▶ Run Video Detection

python detect_video.py

## ▶ Run Webcam Detection

python webcam_detection.py

## 📊 Sample Output
- Person (0.92)
- Car (0.88)
- Dog (0.85)
